MATH513 (MDSD5 2001A, MDSE2 2001A) {UOPMDISFEB2021}R Documentation

Visualizing 2012 Presidential Dialogue

Description

Dataset of the three 2012 presidential debate between Obama and Romney, hosted by Schieffer, Crowley, and Lehrer with some questions from the crowd.

Format

Sample Data Sets as well as data frame prerequisite for package:
Data.frame with 2912 rows and 3 columns:
Person: speaker name
Dialogue: dialogue from the speaker
Turn: Integer field for dialogue sequence
