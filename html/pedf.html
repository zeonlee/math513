The 2012 US presidential debate dataset

Description

Dialogue data from the 2012 preseidential debate. Dataset is stored as Rdata file and stored on https://github.com/zeonlee/math513

Usage

pedf

Format

A dataframe with 3 columns, Person, Dialogue, Turn: #'

Person

Speaker of the dialogue

Dialogue

full dialogue from the speaker

turn

Integer field, or sequence of the dialogues exchanged
